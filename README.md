# Atividade de JavaScript

Este repositório contém exercícios de lógica de programação desenvolvidos em JavaScript.

## Exercícios

### Exercício 1 – Jogo de Adivinhação
O computador gera um número aleatório entre 1 e 20 e o usuário precisa adivinhar o número.

Arquivo: exercicio1.js

---

### Exercício 2 – Pedra, Papel ou Tesoura
O usuário escolhe pedra, papel ou tesoura e o computador escolhe uma opção aleatória.

Arquivo: exercicio2.js

---

### Exercício 3 – Tabuada
O programa pede um número ao usuário e mostra a tabuada de 1 a 10.

Arquivo: exercicio3.js

---

### Exercício 4 – Triângulo de Asteriscos
O programa desenha um triângulo usando * no console.

Arquivo: exercicio4.js

---

### Exercício 5 – Soma da Série
Calcula a soma da série:

1 + 11 + 111 + ...

Arquivo: exercicio5.js
